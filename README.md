PPlayer 基于Ffmpeg4.0.2(arm64-v8a)静态库搭建一套高可用的播播放器
===
>ffmpeg 4.0.2静态库从0开始一个播放器的搭建，支持rtmp、rtsp、hls、本地MP4文件播放，视频解码+音频解码+音视频同步
